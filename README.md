En este trabajo práctico los estudiantes desarrollarán un sitio web de comercio electrónico
enfocado en una empresa importadora de productos. El proyecto se construirá de manera
incremental a lo largo de tres etapas, cada una con requisitos específicos que se deben versionar
en GitHub mediante commits claramente identificados.
El objetivo es integrar el uso de HTML y JavaScript aplicados a un caso real, practicando buenas
prácticas de desarrollo y control de versiones desde el inicio del proyecto
